# *与&
* 为解引用操作符,解除引用，返回的是地址指向的值，取值
& 为取地址操作符，用在某个对象上时，返回的是地址值，取地址


    int val = 1024;  声明一个整数val是1024 
    int *pi2 = new int(1024)  声明pi2为 int(1024)变量指向的地址
